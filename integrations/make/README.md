# WebIntel Make Integration

Connect WebIntel to your Make (formerly Integromat) scenarios.

## Modules

### Actions
- **Scrape Webpage** — Scrape URL to markdown
- **Extract Data** — AI extraction with JSON Schema
- **Lookup Brand** — Brand intelligence for a domain
- **Search Web** — Web search
- **Crawl Domain** — Start a crawl
- **Identify Transaction** — Identify merchant from bank descriptor
- **Classify Business** — NAICS/EIC classification
- **Get Crawl Status** — Check job status
- **List Monitors** — List change monitors
- **Get Brand Logo** — Brand logo URL
- **Get Brand Colors** — Brand color palette

## Setup
1. Get your WebIntel API key
2. Add it as the API Key in the WebIntel connection settings
3. Add modules to your scenarios
