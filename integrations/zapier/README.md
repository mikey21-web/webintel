# WebIntel Zapier Integration

Connect WebIntel to your Zapier workflows.

## Triggers
None yet.

## Searches
- **Lookup Brand** — Get brand intelligence for a domain

## Actions
- **Scrape Webpage** — Scrape URL to markdown
- **Extract Data** — AI extraction with optional JSON Schema
- **Crawl Domain** — Start a crawl job
- **Search Web** — Web search
- **Identify Transaction** — Identify merchant from bank descriptor

## Setup
1. Get your WebIntel API key from the dashboard
2. Add it as your API Key in the WebIntel integration
3. Start building Zaps!
